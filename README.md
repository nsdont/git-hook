git-hook
========

flake8, ctags

### How to use
#### Step 1 
Move config dir to User home dir.
Example:
```
cp -r config ~/.config
```

#### Step 2
Move hook dir to User home git-templates.
Example:
```
mkdir -p .git-templates
cp hook ~/.git-templates/
```

#### Step 3
In your repo dir:
```
git init
```
echo:
Reinitialized existing Git repository in /XX/XX
